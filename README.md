# FreestyleDB

Trick database for freestyle highlining. Includes user accounts to track progression/see insights and trick pages with information and videos.

# Features

So far we have all these cool features with more to come!

## Trick Database

Sortable and filterable view of all submitted freestyle tricks. Select any trick to get a description, tips, and video!

## User Accounts

Your progress is saved to your account so you can access it from both your phone and PC!

## Training Studio

For certain tricks you can access a video player with slowmo playback speeds, no ads, fast loading, and frame-by-frame support.

Editors can add annotations to certain parts of the video to explain cues and tips throughout the video.

Videos are saved to your device cache after watching so they load fast the next time you need them.

## Stack

- **Flutter** — App
- **Supabase** — Database
- **Resend** - Emails
- **Netlify** - Hosting
- **Bunny** - Videos/CDN
