# Filtering
In the filter window we want users to be able to filter by:
- Name
- Difficulty Tier
- Start Pos
- End Pos
- Status (Never Attempted, Attempting, Landed at least once)
- Year landed
- Original performer
# Specifications
Here are the hopes for each filter:
# Name
Show all tricks with partial matches.
## Difficulty Tier
Slider with a start and end tier
## Start and End pos
Two dropdowns to save space, must have an "Any" option by default until a specific position is set
## Status
If a user has never attempted a trick it wil go in that category. If they are attempting it but have not landed it that is a category. If they have landed it at least once (any consistency beyond that) it falls under landed.
## Year Landed
Dropdown with full years displayed.
## Original performer
Basic string search
