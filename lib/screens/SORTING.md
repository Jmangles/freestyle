# Sorting
I want users to be able to change the home screen sorting by a few metrics.
# Specifications
There should be a Sort By tab where users can choose how tricks are sorted.
A primary sort type sorts the page with headers like we currently do with Difficulty
A secondary sort type sorts the tricks within each group by that metric
There should be support for ascending/descending order.
## Primary Sort Types
- Difficulty Tier
- Start Position
- Year first landed
- Consistency
## Secondary Sort Types
- Difficulty Tier (does nothing if difficulty tier is the primary)
- Start Position
- End Position
- Consistency
- Alphabetical
